# BOARDRAW

An online open source whiteboard app with video chat features.


## Deployment

To deploy this project run

```bash
  npm start
```


## Features

- Special Grids for whiteboard
- Easily Accessable
- Fullscreen mode
- Download draw templates



## Running Tests

To run tests, run the following command

```bash
  npm run test
```


## Tech Stack

**Client:** React, Redux, TailwindCSS, JS

**Server:** Node, Express, JS

## Run Locally (White board)

Clone the project

```bash
  git clone https://github.com/VaibhavUpreti/BOARDRAW.git
```

Go to the project directory

```bash
  cd BOARDRAW
  cd board 
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
